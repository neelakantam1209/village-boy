# **App Name**: UrbanLink

## Core Features:

- Service Catalog: Display a catalog of available services with details like pricing, descriptions, and customer reviews.
- Location-Based Services: Dynamically display services based on the user's selected location using a location dropdown selector.
- Booking System: Enable users to pick a service, sub-service, choose a date and time, add their address, and confirm the booking.
- Admin Dashboard: Provide an admin panel for managing services, categories, users, and orders, including adding, editing, and deleting content.
- AI-Powered Service Recommendations: Generative AI tool to offer personalized service recommendations based on user behavior and preferences. LLM is the reasoning tool.
- Search System: Implement a search functionality enabling users to search services by title, category, or location.
- User Authentication: Implement JWT authentication for both users and admin to secure the application.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to convey trust and professionalism.
- Background color: Light Gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#008080) to highlight key interactive elements.
- Body and headline font: 'Inter', a sans-serif font that provides a modern, neutral look, suitable for both headlines and body text.
- Use consistent and clear icons to represent service categories and actions.
- Responsive layout optimized for various screen sizes, ensuring usability on all devices.
- Subtle animations for transitions and interactive elements to enhance user experience.
